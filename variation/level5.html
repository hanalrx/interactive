Restrictions


  • Use both HTML and CSS

  • Use any HTML element (except for img, iframe, and a via linking to external sites. This is a purely typographic project, so no external images, links, or other assets should be used.)

  • In addition to the CSS properties allowed in Level 3, you can use any of the following CSS properties:
    • font-style
    • font-weight
    • font-size
    • font-family
    • text-align
    • line-height
    • letter-spacing
    • text-transform

  • Again, you can use & n b s p ; (but without spaces) to equal a visible space.

Goals


Focus on typography using CSS. Experiment with hierarchy and scale.
